# cryptographic-protocols
Лабораторные работы по дисциплине "Криптографические протоколы".
<br>Лабораторная работа №1. Алгоритм шифрования Twofish.
<br>Лабораторная работа №2. Алгоритм хеширования BMW.
<br>Лабораторная работа №3. Алгоритм ЭЦП BLS.
<br>Лабораторная работа №4. Режимы шифрования ECB, CBC, CTR-ACPKM, MGM, OFB, OMAC.
<br>Лабораторная работа №5. ПДСЧ.
<br>Лабораторная работа №6. Алгоритм диверсификации ключа.
<br>Лабораторная работа №7. Протокол взаимной аутентификации с использованием(протокол Нидхема — Шрёдера) и без использования(протокол SRP) доверенной третьей стороны.
<br>Лабораторная работа №8. Протокол защищённого обмена сообщениями.
<br>Для тестирования работы с файлами нужно указать существующие пути.
